# Estudos — HTML5 & CSS3

Este repositório reúne **todos os módulos, exercícios e projetos práticos** desenvolvidos durante a conclusão do **curso completo de HTML5 e CSS3**, ministrado por **Gustavo Guanabara**, na plataforma **Curso em Vídeo**.

O conteúdo aqui representa minha base sólida em **desenvolvimento front-end**, construída com foco em **boas práticas, organização de código e responsividade**.

## Conteúdos abordados
Ao longo do curso, foram estudados e aplicados os seguintes conceitos:


### HTML5
- Estruturação de páginas
- Tags semânticas
- Listas, tabelas e formulários
- Imagens, áudio e vídeo
- Boas práticas de acessibilidade


### CSS3
- Estilização com cores e tipografia
- Box Model
- Flexbox
- Grid Layout
- Media Queries
- Layouts responsivos
- Organização e reutilização de estilos
